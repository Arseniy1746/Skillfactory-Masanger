# NewMasanger
Список участников команды: Арсений(Я)

#Идея решения
Я разработал консольный мессенджер, который работает полностью в терминале, не требуя графического интерфейса
Принципы:
Простота использования — интуитивно понятное меню с цифровым выбором действий
Типы общения — разделение на публичные (общий чат) и приватные (личные) сообщения
Безопасность — авторизация по ID и паролю
Надёжность — защита от некорректного ввода и переполнения памяти

#class User
Представляет зарегистрированного пользователя чата.

Метод	Описание
User(name, password, id)	Конструктор, создаёт нового пользователя
SetName(newName)	Изменяет имя пользователя
GetName()	Возвращает имя пользователя
GetPassword()	Возвращает пароль
GetID()	Возвращает уникальный ID пользователя

#class Message
Представляет сообщение в чате (публичное или личное).

Метод	Описание
Message(from, text, to)-Конструктор, создаёт сообщение
GetFrom()-Возвращает имя отправителя
GetText()-Возвращает текст сообщения
GetToID()-Возвращает ID получателя (-1 = всем)
IsPrivate()-Проверяет, является ли сообщение личным
Print()-Выводит сообщение в консоль

#template<typename T, int MAX_COUNT> class SimpleArray
Шаблонный класс-контейнер с фиксированным размером.

Метод	Описание
Add(item)-Добавляет элемент (принимает unique_ptr)
Size()-Возвращает текущее количество элементов
Get(index)-Возвращает указатель на элемент по индексу
Clear()-Очищает массив
